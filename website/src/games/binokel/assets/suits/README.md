# Farbsymbole (Trumpf)

Platzhalter-Bilder für die vier Binokel-Farben, gezeigt auf den Trumpf-Buttons.

- `eichel.webp`, `blatt.webp`, `herz.webp`, `schellen.webp`
- Quadratisch (aktuell 160×160), WebP.
- Einfach durch die echten Symbole ersetzen - gleiche Dateinamen behalten, dann
  greift alles automatisch (statischer Import in `../suit-images.ts`).
